# Aadith Santosh | Independent Equity Research

Static GitHub Pages site for publishing equity research reports and financial analysis.

## Structure

- `index.html` - homepage
- `about/index.html` - analyst profile and research philosophy
- `research/index.html` - research archive
- `research/ihcl-2026.html` - first report page template
- `assets/css/style.css` - global styling
- `assets/images/` - images and charts
- `assets/pdf/` - downloadable report PDFs
- `sitemap.xml` and `robots.txt` - SEO basics

## Publishing

This site is designed for a GitHub Pages user site at:

```text
https://www.aadithsantosh.com/
```

If this folder is connected to the `AadiZer01/aadithsantosh.github.io` repository, commit and push these files to the default branch. GitHub Pages will serve `index.html` from the repository root.
